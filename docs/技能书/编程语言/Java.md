# Java

## 基本语法

单行注释：`// 注释内容` 
多行注释：`/* 注释内容 */`
赋值：`int x = 1;`
加减乘除：`int x = a + b*b - a/b;`
三目运算：`int x = (a < b) ? a : b;`
流程控制1：`  if (a) { } else if (b) { } else { }`
流程控制2：`  switch (x) { case 1: { }; break; case 2: { }; break; default: { } }`
循环1：`for(int i=0; i < n; i++) { /* 判断->执行->递增 */ }`
循环2：`while(x) { /* 先判断再执行 */}`
循环3：`do { /* 先执行再判断 */} while(x)`
